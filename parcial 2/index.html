Mis Proyectos

En mis proyectos busco combinar creatividad, tecnología y solución de problemas. Aquí algunos de los proyectos que estoy desarrollando:

Proyecto 1: Aplicación Web Interactiva

Este proyecto consistió en el desarrollo de una aplicación web interactiva que permite a los usuarios gestionar tareas de manera sencilla y efectiva.

  • Tecnologías utilizadas: HTML, CSS, JavaScript, Node.js
  • Objetivo: Facilitar la organización personal mediante una interfaz amigable.

Proyecto 2: Sistema de Gestión de Tareas

En este proyecto, diseñé un sistema de gestión de tareas para pequeñas empresas utilizando Java para el backend y MySQL para la base de datos.

  • Tecnologías utilizadas: Java, MySQL, Spring Boot
  • Objetivo: Optimizar la administración de proyectos y tareas en pequeñas empresas.

Proyecto 3: Videojuego Educativo

El concepto de este proyecto es un videojuego educativo que mezcla entretenimiento y aprendizaje, orientado a enseñar matemáticas básicas a niños.

  • Tecnologías utilizadas: Unity, C#
  • Objetivo: Facilitar el aprendizaje de manera divertida y dinámica.
